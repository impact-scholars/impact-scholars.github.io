# Impact Scholars Gallery

This site collects the micropublications produced by participants in the [Impact Scholars Program](https://neuromatch.io/impact-scholars-program/) (ISP), a six-month, part-time mentored research initiative run by [Neuromatch](https://neuromatch.io).

Scholars from Neuromatch Academy's courses in Computational Neuroscience, Deep Learning, NeuroAI, and Computational Tools for Climate Science work in small groups with expert mentors to produce original research. Each project concludes with a citable micropublication, openly published here.

Browse publications by cohort year using the sidebar, or start with the [2023 Edition](impact-scholars/impact-scholars-2023.md).

Want to learn more about the program? Visit the [Impact Scholars Program](impact-scholars.md) page or head to [neuromatch.io](https://neuromatch.io/impact-scholars-program/) for full details.
